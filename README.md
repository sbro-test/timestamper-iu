## INSTALLATION UND START
  * entwickelt auf Linux (Debian11), getestet auf Windows7 und Windows10
  
  * Python3 Interpreter ist erforderlich
  * wxPython muss mit "pip install wx" installiert werden
  
  * gestartet wird
    `python3 timestamper_gui.py`
    
    *(auf Wunsch mit einem Parameter für das Startverzeichnis)*
